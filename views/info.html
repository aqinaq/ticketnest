About TicketNest

TicketNest is a ticket booking demo built with Express.js. It shows routing, forms, middleware, and an API-driven frontend.

Search with query params CRUD via API Server-side POST form

Features

  • Search events using query parameters
  • View + manage events in one page
  • Send messages via a contact form
  • Read events from JSON API
